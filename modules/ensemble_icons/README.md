# Ensemble Icons

Icons for Ensemble

## Usage

To use the icons in your project, you can use the following code:

```yaml
- Icon:
    name: "icon-name"
    library: "icon-library" # default / remix / fontAwesome
    size: "icon-size"
    color: "icon-color"
```

## Update Icon Library

To update the icon library follow the documentation in [update folder](/update/)
